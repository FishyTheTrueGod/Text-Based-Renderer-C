# Terminal Bouncing Star in C

A pure C terminal engine that renders a bouncing star with smooth frame pacing and edge collision detection. I built this from scratch to skip beginner code and learn low-level logic, memory management, and how engine architecture works visually. Project 1 of an 8-project roadmap toward embedded systems.

## How to Run It
```bash
gcc main.c -o bouncing_star
./bouncing_star
```
